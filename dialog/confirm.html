确认提示

替代默认的确认提示 点击查看演示

How to use?

引入以下资源

<script src="../libs/jquery-1.11.1.min.js" type="text/javascript"></script>
<script src="./jquery-confirm.js" type="text/javascript"></script>
<link rel="stylesheet" href="./alert.css">

如何调用

$.confirm({ title:'This is alert title' // 标题
 content:'This is alert content'// 内容
 sure: callback // 用户按下确定后的回调函数
 cancel: callback // 用户按下取消的回调函数
});

Tip:确认提示组件无法被同时触发，如果页面上有该组件，其他触发动作将被阻止

为了便于风格统一 Alert 组件和 Confirm 组件使用相同的皮肤。
