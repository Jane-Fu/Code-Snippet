消息提示

替代默认alert的消息提示 点击查看演示

How to use?

引入以下资源

<script src="../libs/jquery-1.11.1.min.js" type="text/javascript"></script>
<script src="./jquery-alert.js" type="text/javascript"></script>
<link rel="stylesheet" href="./alert.css">

基本调用

$.alert('This is an alert');

其他参数

$.alert({ title:'This is alert title' // 标题
 content:'This is alert content'// 内容
 timeout: 1 // 是否自动隐藏提示框
});

Tip:多个消息提示窗口会层叠显示

为了便于风格统一 Alert 组件和 Confirm 组件使用相同的皮肤。
